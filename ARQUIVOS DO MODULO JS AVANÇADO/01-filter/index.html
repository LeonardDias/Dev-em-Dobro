Filter

O método filter em JavaScript é usado em arrays para criar um novo array com todos os elementos que passam em um teste fornecido por uma função. Esse método não modifica o array original; ele apenas retorna uma nova versão dele com os elementos que atendem ao critério especificado.

Confira os comentários no código e o Console através do Inspecionar (F12) para entender o conteúdo! ^-^
