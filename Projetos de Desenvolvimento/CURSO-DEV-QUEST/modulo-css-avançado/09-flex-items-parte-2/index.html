Flex

A propriedade FLEX em flexbox é uma propriedade abreviada que combina as propriedades FLEX-GROW, FLEX-SHRINK, e FLEX-BASIS em uma única declaração.

O padrão do FLEX é o padrão innicial de cada um dos três nesta ordem: GROW, SHRINK e BASIS, ou seja 0 1 0.

Geralmente utilizamos o FLEX: 0 (EVITA QUE TODOS OS FLEX-ITEMS FILHOS OCUPEM TODO O TAMANHO DO CONTAINER PAI) e o FLEX: 1 (E AQUI EXPANDE TODOS OS FLEX-ITEMS FILHOS PARA OCUPAR TODO O ESPAÇO DO CONTAINER PAI).

Order

A propriedade ORDER é usada para controlar a ordem de exibição dos elementos filhos (FLEX-ITEMS) dentro de um container flexível (elemento pai) quando eles são renderizados na tela. Essa propriedade permite que você reorganize visualmente a ordem dos itens sem alterar a estrutura do HTML.

Neste caso como você pode ver mudamos o item 3 com o ORDER no valor de 1, se eu quiser jogar o item 1 para o final depois do 3 o ORDER tem que ser 2. E assim por diante.

Por padrão todos os FLEX-ITEMS (filhos) dentro de um CONTAINER (pai) tem por padrão o ORDER: 0;, e como estamos por padrão também no FLEX-DIRECTION: ROW, se modificarmos o ORDER para 1 logo o FLEX-ITEM escolhido irá para o final. Se colocarmos o FLEX-DIRECTION: ROW-REVERSE no CONTAINER, ele irá para o inicio.

Align Self

A propriedade align-self é usada para controlar a posição vertical de um único item flex dentro de um container flexível. Ela substitui a configuração de alinhamento vertical definida pelo container para um item específico, permitindo que você ajuste a posição vertical de um item individualmente, sem afetar os outros itens no mesmo container. Isso é especialmente útil quando você deseja que um item se comporte de forma diferente dos demais.
