Flex Items - Grow - Basis - Shrink

São os elementos HTML filhos dentro do contêiner flex. Estes são os itens que serão organizados e distribuídos de acordo com as regras do flexbox. Cada item individual é referido como um "flex item".

Grow

A propriedade "flex-grow" define a proporção na qual um item flexível cresce em relação aos outros itens flexíveis no mesmo contêiner. Ela aceita um valor numérico, que representa a proporção de crescimento. Quanto maior o valor, mais o item crescerá em relação aos outros. Por padrão, todos os itens têm um valor de "flex-grow" igual a zero, o que significa que eles não crescerão e ocuparão apenas o espaço disponível para seu conteúdo.

Basis

A propriedade flex-basis é usada em Flexbox para definir o tamanho inicial de um item flexível dentro de um contêiner flexível. Ela especifica o tamanho base que um item flexível deve ter antes de qualquer distribuição de espaço extra ou encolhimento de espaço ocorrer, com base nas propriedades flex-grow e flex-shrink.

Shrink

A propriedade flex-shrink é usada em flexbox para determinar como os itens flexíveis dentro de um contêiner flexível devem diminuir de tamanho quando há espaço insuficiente no contêiner para acomodar todos os itens com seus tamanhos iniciais.
