Async e Await

ASYNC = A palavra-chave async é usada para declarar uma função assíncrona. Isso significa que a função sempre retornará uma Promise. Mesmo que você não retorne explicitamente uma Promise, a função automaticamente se transforma em uma. Se você retornar um valor, ele será encapsulado em uma Promise resolvida, e, se houver um erro, a Promise será rejeitada.

AWAIT = A palavra-chave await só pode ser usada dentro de uma função async e é usada para esperar que uma Promise seja resolvida antes de continuar a execução do código. É como se você estivesse "pausando" a execução da função até que a Promise retorne um valor. Isso evita a necessidade de usar .then() e faz o código parecer síncrono.
