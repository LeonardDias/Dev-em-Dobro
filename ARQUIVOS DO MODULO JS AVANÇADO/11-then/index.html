Then

O método .then() em JavaScript é usado em Promises, uma estrutura que permite lidar com operações assíncronas (como carregar dados de uma API) de maneira mais organizada. O .then() é chamado assim que uma Promise é resolvida (ou seja, quando a operação termina com sucesso).

Confira os comentários no código e o Console através do Inspecionar (F12) para entender o conteúdo! ^-^
