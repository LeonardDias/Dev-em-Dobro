Position Fixed

Aqui iremos aprender a colocar um HEADER com propriedade POSITION e valor FIXED, isso vai deixar uma BARRA / MENU / ETC... travado na tela, mesmo que você role a tela pra baixo ela continuará aparecendo. Colocaremos também dentro do HEADER uma TAG NAV (utilizada para menus de navegação) e dentro dela UL > LI.
