Flex Container: Direction - Wrap - Flow

Direction

A propriedade flex-direction determina a direção em que os itens flexíveis são posicionados dentro do contêiner flex.

Wrap

A propriedade flex-wrap determina se os itens flexíveis devem ser dispostos em uma única linha (valor padrão) ou em várias linhas, se o espaço for insuficiente.

Flow

A propriedade flex-flow é uma abreviação para definir simultaneamente as propriedades flex-direction e flex-wrap. Ela aceita um valor combinado dessas duas propriedades. O formato é {flex-direction} {flex-wrap}.
