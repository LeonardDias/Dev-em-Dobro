Flex Container - Justify content

Aqui iremos aprender sobre a PROPRIEDADE JUSTIFY-CONTENT e seus VALORES.

A propriedade justify-content é usada para alinhar os itens (FLEX-ITENS) dentro de um contêiner flexível ao longo do eixo principal (main axis). O eixo principal pode ser horizontal (quando a flex-direction é row) ou vertical (quando a flex-direction é column). Existem várias opções de alinhamento disponíveis para a propriedade justify-content.

O JUSTIFY-CONTENT só irá alinhar os FLEX-ITENS HORIZONTALMENTE.

Aqui iremos visualizar os FLEX-ITENS, então fique atento que o JUSTIFY-CONTENT irá apenas o que esta dentro do quadrado LARANJA, para alinhar o QUADRADO AZUL você utilizara o FLEX-DIRECTION (ROW, ROW-REVERSE, COLUMN, COLUMN-REVERSE), compare sempre a imagem acima à explicação abaixo para entender melhor.

Flex-start

Alinha os flex-itens no início do eixo principal dentro do CONTAINER. Esse é o valor PADRÃO e tendo o FLEX-DIRECTION no ROW irá começar da ESQUERDA para a DIREITA.
