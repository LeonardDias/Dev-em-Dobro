Flex Container - Align items

A propriedade align-items é usada para controlar a posição vertical dos elementos dentro de um contêiner flexível (um flex container). Ela define como os itens flexíveis são alinhados ao longo do eixo transversal (eixo Y) em relação ao contêiner flexível.

Aqui iremos continuar a controlar os FLEX-ITEMS, utilizando o ALIGN ITEMS e com isso movimentar o CONTEÚDO VERTICALMENTE. Com isso entendemos que para movimentar os FLEX-ITENS usaremos o JUSTIFY-CONTENT para alinhar eles HORIZONTALMENTE e o ALIGN ITENS alinha VERTICALMENTE.

Flex-start

Este é o valor padrão. Ele alinha o conteúdo dos flex-itens no início do eixo transversal do contêiner, empurrando-os para o topo do contêiner, se houver espaço extra.
