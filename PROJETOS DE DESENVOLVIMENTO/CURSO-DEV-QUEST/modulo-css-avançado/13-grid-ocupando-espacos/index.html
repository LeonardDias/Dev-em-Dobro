GRID - Ocupando Espaços

Aqui iremos aprender a ocupar os espaços do GRID assim como a imagem abaixo.

Agora iremosa definir na CSS algumas PROPRIEDEADES e VALORES para deixar desta forma. Utilizamos o display: grid; (para transformar em grade), grid-template-columns: repeat(5, auto); (para dar 5 colunas em tamanho auto), grid-template-rows: repeat(3, auto); (para dar 3 linhas de tamanho auto), gap: 15px (para dar um espaçamento entre as células de 15px para todas as direções);

Maiores detalhamentos no NOTION.
