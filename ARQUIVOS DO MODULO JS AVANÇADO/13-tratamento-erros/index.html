Tratamento de Erros no Javascript

Para tratar erros em JavaScript, usamos principalmente o bloco try...catch. Isso permite executar um trecho de código (try) e, caso ocorra um erro, capturá-lo no bloco catch. Também existe a opção de usar o finally, que executa código independentemente de ter ocorrido um erro ou não.

Confira os comentários no código e o Console através do Inspecionar (F12) para entender o conteúdo! ^-^
