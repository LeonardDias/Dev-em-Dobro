Promises

No JavaScript, uma Promise (ou promessa) é um objeto que representa um valor que pode estar disponível agora, no futuro ou nunca. As Promises são usadas para gerenciar operações assíncronas, como carregar dados de uma API ou esperar por um tempo específico, permitindo que o código continue executando enquanto a operação ainda está em progresso.

Confira os comentários no código e o Console através do Inspecionar (F12) para entender o conteúdo! ^-^
