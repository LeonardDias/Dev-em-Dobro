Desestruturação de OBJETOS e ARRAYS

A desestruturação é uma maneira de extrair dados de arrays ou objetos e armazená-los em variáveis de forma mais concisa. Esse recurso facilita o acesso a valores específicos sem precisar escrever várias linhas de código. Vamos ver exemplos de desestruturação em objetos e arrays.

Confira os comentários no código e o Console através do Inspecionar (F12) para entender o conteúdo! ^-^
