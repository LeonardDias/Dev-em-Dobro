Display


Display block

Aqui temos como exemplos os DISPLAY: BLOCK (que é o padrão), ele vai ocupar toda a linha e jogar o próximo item para baixo. Chamado também de BLOCK LEVEL, você pode mudar as propriedades dele colocando largura para vizualizar melhor, mas não irá mudar o display, assim como vemos no item 1, ele tem uma largura de 100px porém continua display:block. Alguns exemplos de TAGS BLOCK LEVEL: DIV, P, H1 ao H6, UL > LI, OL > LI, TABLE, FORM, etc.

Display inline

Aqui temos como exemplos os DISPLAY: INLINE, ele não vai ocupar toda a linha e irá jogar o próximo item ao lado. Chamado também de INLINE BLOCK, você não pode mudar as propriedades dele, pois ele irá apenas se adequar ao conteúdo, porém você pode fazer outras configurações como MARGIN e PADDING, mas não irá mudar o display, assim como vemos no item 1 ele se adequando ao tamanho digitado. Alguns exemplos de TAGS INLINE LEVEL: SPAN, A, STRONG, EM (texto enfatizado), IMG, BR, ABBR, etc.

Display inline-block

Aqui temos como exemplos o DISPLAY: INLINE-BLOCK, que como o valor já diz ele é uma fusão de ambos os exemplos informados acima, juntando as caracteristicas de cada um. Ele irá colocar os itens em INLINE e você irá escolher à altura e largura do BLOCK. como vemos no exemplo a baixo.

Display none

E por ultimo é o DISPLAY: NONE, que esconde o conteúdo. Basta olhar no arquivo da CSS que você irá ver que ele está lá. ATENÇÃO ele está lá mas o usuário não irá ver!
