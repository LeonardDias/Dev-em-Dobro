Cantralização Vertical

Aqui estamos aprendendo a na vida real um FLEX-ITEM, pois anotigamente se utilizava muito o TRANSFORM: TRANSLATE, e hoje com as mudanças na CSS você pode utilizar no CONTAINER (PAI) o DISPLAY: FLEX nos FLEX-ITEMS (FILHO) para ficar de forma mais limpa. OBS: Mas nada lhe impede de usar o TRANSFORM ok!

Então aqui é o seguinte para colocarmos os FLEX-ITEMS centralizado basta colocar no H2 as seguintes propriedades na CSS: DISPLAY: FLEX, JUSTIFY-CONTENT: CENTER (para alinhar horizontalmente no centro da linha) e ALIGN-ITENS: CENTER (para alinhar Verticalmente no centro).
