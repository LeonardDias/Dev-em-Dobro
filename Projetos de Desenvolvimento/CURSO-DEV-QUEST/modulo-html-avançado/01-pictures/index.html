Aqui iremos aprrender a colocar as imagens responsivas ao tamanhos de telas dos usuários, para isso iremos utilizar a TAG PICTURE e dentro dela o SOURCE (para os tamanhos de telas desejadas) e o IMG (para a menor imagem escolhida).

Então basta colacar dentro de PICTURE os SOURCES e a IMG, sendo que respeitando a ordem de primeiro os SOURCES de maior para a menor e depois a IMG.
