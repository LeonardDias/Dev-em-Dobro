Flex Container: Direction - Wrap - Flow

Aqui iremos começar a aprender o ná prático o FLEXBOX e suas PROPRIEDADES.

Analisaremos todo o conteúdo desta aula comparando com a imagem acima, para ficar mais fácil a compreensão.

Direction

Nossa primeira PROPRIEDADE é o DIRECTION, para começar temos um CONTAINER PAI que neste caso aqui é a TAG FIELDSET com CLASS = CONTAINER1 e DIRECTION e dentro dela temos os FILHOS que são outras DIVS com CLASS = ITEM.

A propriedade flex-direction determina a direção em que os itens flexíveis são posicionados dentro do contêiner flex. E existem 4 diferentes VALORES para essa PROPRIEDADE:

ROW - Com esse valor temos o alinhamento dos itens na linha (PADRÃO) da ESQUERDA para a DIREITA. Começando do INICIO PRINCIPAL para o FIM PRINCIPAL do CONTAINER.

ROW-REVERSE - Com esse valor temos o alinhamento dos itens na linha também, porém eles serão alinhados da DIREITA para a ESQUERDA. Começando do FIM PRINCIPAL para o INICIO PRINCIPAL do CONTAINER.OBS: No exemplo criado da aula os quadrados possuem números e com este exemplo fica do 3 para o 1.

COLUMN - Com esse valor teremos o alinhamento dos itens em colunas (PADRÃO), de CIMA para BAIXO, do INICIO TRANSVERSAL para o FIM TRANSVERSAL da imagem.

COLUMN-REVERSE - Com esse valor teremos o alinhamento dos itens em colunas também, porém eles serão reversos de BAIXO para CIMA, do FIM TRANSVERSAL para o INICIO TRANSVERSAL da imagem. OBS: No exemplo criado da aula os quadrados possuem números e com este exemplo fica do 3 para o 1.

Wrap

A propriedade FLEX-WRAP determina se os itens flexíveis devem ser dispostos em uma única linha (valor padrão) ou em várias linhas, se o espaço for insuficiente. Ele é usado com FLEX-DIRECTION = ROW. Ele contém 3 principais PROPRIEDADES, que são:

WRAP - Com esse valor quando a tela for ficando menor ele irá começar a descer os FLEX ITENS da linha e irá manter o tamanho dos itens e descendo, deixando assim os itens alinhados até que vire uma coluna com um item embaixo do outro.

WRAP-REVERSE - Com esse valor quando a tela for ficando menor ele irá começar a descer os FLEX ITENS da linha, porém os FLEX ITENS serão revertidos (em vez de começar do 1 ao 6 ele manterá em cima do 6 ao 1) e irá manter o tamanho dos itens e descendo, deixando assim os itens alinhados até que vire uma coluna com um item embaixo do outro.

NOWRAP - Com esse valor quando a tela for ficando menor ele manterá os FLEX ITENS na mesma linha e irá reduzir o tamanho dos itens para continuar o alinhamento.

Flow

A propriedade flex-flow é uma abreviação para definir simultaneamente as propriedades flex-direction e flex-wrap. Ela aceita um valor combinado dessas duas propriedades. O formato é {flex-direction} {flex-wrap}.

Neste caso é a junção de ambas as PROPRIEDADES DIRECTION e WRAP, em uma só.
