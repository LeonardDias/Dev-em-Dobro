Variaveis

Aqui iremos falar de algo muito importante que são as VARIÁVEIS, com elas armazenamos valores para utilização na CSS, facilitando muito e nos ajudando de formas mais simples. Um exemplo é, imagine trocar uma cor em 100 PROPRIEDADES na CSS (TRABALHOSO), agora imagina que você armazenou essa cor em uma VARIÁVEL, basta apenas trocar nela que as outras 100 cores irão mudar automaticamente.

ATENÇÃO: até a data de hoje 17/11/2023 existem alguns navegadores que não aceitam VARIÁVEIS, que são os INTERNET EXPLORER e o OPERA MINI. Sempre averigue no site do CAN I USE e digite VARIABLES para mais detalhes.

Existem dois tipos de VARIÁVEIS as de escopo GLOBAIS (que podem ser utilizadas em todo o documento neste caso é criado a RAIZ na CSS que é a ROOT:) e as de escopo LOCAIS (que só podem ser usadas nos seletores em que elas foram criadas.). EX: :root {--cor-padrao: green; --tamanho-de-fonte-padrao: 30px;}.

Aqui iremos estilizar a CSS com as seguintes propriedades: h2 {--cor-padrao: blue; color: var(--cor-padrao, red); font-size: var(--tamanho-de-fonte-padrao, 20px); background-color: red;}. Isso significa que a cor padrão da fonte será azul, se der algum erro ela ira receber a cor padrão da root verde e se der errado vermelho, a cor de fundo será vermelha e o tamanho da fonte será 20px.

Aqui neste caso o --cor-padrao: blue; foi criado como uma VARIÁVEL LOCAL, ou seja apenas dentro da seletor H2.;
