Seletores

Vamos dar uma olhada agora nas configurações dos SELETORES. E aqui já podemos configurar o H1 na CSS ficando desta forma: h1 {color: green;}. Preste atenção pois escolhendo apenas SELETORES de TAGS você irá configurar TODAS as TAGS H1 da página. Por isso usamos mais ID e CLASS, pois podemos dar nomes a elas e configurar individualmente.

Estilização com Classes CLASS

Aqui iremos aprender a classificar as CLASS e dar um nome a ela para configurarmos na CSS. Então 1º damos um nome a ela EX: class="paragrafo" e depois na CSS iremos começar a estilização colocando um ponto (.) seguido do nome da CLASS EX: .paragrafo {color: blue;}. Ou seja toda TAG que tiver a CLASS="paragrafo ficará com as letras azul"

Aqui vemos que mesmo a TAG sendo um SPAN elá está azul, pois recebeu a CLASS="PARAGRAFO".

Estilização com ID

Aqui vamos estilizar com ID que basicamente o que muda com relação ao CLASS é que lá você faz uma configuração e coloca em vários TAGS que quiser. Já no ID não, ela é específica pra cada TAG que você quiser configurar. A W3C diz que não é bom colocar ID repetidas e sim utilizar apenas uma ID por página. Isso nos informa que você pode criar varias ID por páginas mas como nomes diferentes. ATENÇÃO: O ID é mais utilizado para o JAVASCRIPT, mas não impede de você utlizar, o conselho é utilize mais CLASS para estilização e ID para JAVASCRIPT.

E para utiliza-las começamos com a # seguida do nome do seletor. EX: #menu {margin-left: 500px;}.

Estilização com duas Classes

Aqui iremos colocar duas classes no mesmo CLASS da seguinte forma, EX: CLASS="titulo vermelho", e neste caso apenas separamos com um espaço. E para a utilização na CSS colocamos desta forma, EX: .titulo.vermelho {color: red;}, veja que na CSS não separamos com espaço e sim com (.) e os nomes tudo junto.

Você também pode averiguar que na CSS existe apenas um seletor chamado de .título com font-size 50px e isso nos diz ambos os H3 irão receber este tamanho pois tem as classes igual. Ou seja em vez de colocar o font-size nas configurações dos dois VERMELHO / LARANJA colocamos apenas no TITULO.

Estilização da página

Podemos fazer uma configuração geral da página utilizando o seletor *, e desta forma configurará toda a página. Neste exemplo CRIAMOS uma CSS chamada RESET e linkamos com o HTML e colocamos lá as seguintes configurações: * {padding: 0; font-size: 30px;} e desta forma está dizendo que todos os elementos da página terá o PADDING ZERO e o tamanho da letra será 30PX.

Podemos também configurar as TAGS HTML, HEAD, BODY separadamente. Atente que se você configurar a TAG BODY o que está em HEAD não receberá a estilização, pois HEAD está fora do BODY.

Estilização com vírgula

Podemos configurar várias TAGS sem CLASS e ID diretamente separando apenas com uma virgula, EX: p, span {font-size: 30px; text-align: justify;}, e neste caso a estilização está nos informado que TODAS as TAGS P e SPAN da página irão receber o tamanho da fonte de 30px e o texto alinhado justificado.

Você também pode confirgurar qualquer título de seletor desta forma, basta seguir as configurações. Isso evita criar menos código para configurações iguais e sebemos que menos códigos é melhor.

Estilização de classe pai com classe filho

Neste exemplo iremos estilizar as CLASS de forma parecida com a Estilização com duas Classes, sendo que aqui como uma class está dentro da outra separamos os seletores com um espaço ficando da seguinte forma: .sobre .texto {color: palevioletred;} e o texto irá ficar na cor palevioletred.

Colocamos a estilização da CLASS SOBRE com uma cor de background de blanchedalmond e na CLASS TEXTO uma cor palevioletred e background color azul e podemos ver que a o backgroundo do filho ficou como estilizado, mas este PARAGRAFO que não recebeu uma CLASS continua com o BACKGROUND da CLASS PAI (SOBRE).

Estilização com Seletor (>)

Aqui quando colocamos o sinal de > ionformamos que estamos selecionando todos os filhos dentro do pai, neste caso temos um DIV pai e dois filhos que são P. Ex: div > p {color: royalblue;} Então logo vemos que todos os P dentro da DIV receberam as configurações de cor royalblue.

Aqui a estilização irá escolher o 1º elemento depois ou seja a TAG P depois da TAG aqui, que neste caso é a DIV, ou seja ela é uma TAG irmã. EX: div + p {color: aqua;}. pois como a DIV abre e fecha com os números e o P abre e fecha com o texto e não estão dentro de um pai, logo são irmãos.

Estilização com atributo

Aqui de forma simples e direta você irá configurar com o ATRIBUTO da TAG. Neste caso escolhemos o TARGET de um e o HREF de outro e as configurções ficaram assim: a[target=_blank] {color: red;} a[href="google.com"] {color: chartreuse;}
