Flex Items - Grow - Basis - Shrink

São os elementos HTML filhos dentro do contêiner flex. Estes são os itens que serão organizados e distribuídos de acordo com as regras do flexbox. Cada item individual é referido como um "flex item".

Aqui iremos explorar o que será setado nas PROPRIEDADES dos FLEX ITEMS como podemos ver a imagem abaixo.

Shrink

A propriedade flex-shrink é usada em flexbox para determinar como os itens flexíveis dentro de um contêiner flexível devem diminuir de tamanho quando há espaço insuficiente no contêiner para acomodar todos os itens com seus tamanhos iniciais.

O PADRÃO do FLEX-SHRINK é 1, não importando o TAMANHO do CONTAINER os FLEX-ITEMS irão se adaptar ao tamanho do CONTAINER.
