Pseudo Elementos

Aqui iremos aprender sobre PSEUDO-ELEMENTOS e algumas de suas propriedades. Ele é utilizado da seguinte forma Ex: seletor::pseudo-elemento {propriedade: valor;}.

ATENÇÃO: pois nem todo PSEUDO-ELEMENTO funciona em TAG estilo INLINE LEVEL, apenas em BLOCK LEVEL. Fique atento e opte pela melhor opção.

Você também pode usar CLASS e ID para estilizar com os PSEUDO-ELEMENTOS.

First Letter

Como o próprio nome diz iremos estilizar apenas a 1ª Letra deste Parágrafo, neste caso aqui colocamos uma color coral e um font-size 100px. .first-letter::first-letter {color: coral; font-size: 100px;}

First Line

Neste segundo exemplo iremos personalizar a 1ª letra e a 1ª linha com o FIRST LINE e combinar ambos. Colocamos as seguintes configurações: .first-line::first-line {color: coral; font-size: 2em;} .first-line::first-letter {color: blue; font-size: 4em;}

Marker

Podemos também usar o MARKER para estilizar os marcadores de uma lista. EX: li::marker {color: red; font-size: 23px;}

Agora iremos aprender os pseudo-elementos (provavelmente) mais utilizados que são o BEFORE e o AFTER, que são respectivamente o antes e o depois e serve para estilizar algo, como veremos a seguir.

Aqui colocaremos um coração (mas você pode colocar o que quiser e estilizar da mesma forma) ANTES do TÍTULO. De uma olhada na CSS que existem algumas formas de configurações SHORTHAND ou mais detalhadas.

Meu Título BEFORE

Meu Título AFTER

As configurações na CSS vai depender das suas escolhas, então preste atenção.

Exemplo que você pode utilizar neste caso de BEFORE e AFTER é em vez de usar uma imagem (img) / texto (p) / etc... no HTML (pois isso pode poluir o HTML) é melhor usar o PSEUDO-ELEMENTO e estilizar com CONTENT na CSS.
