Parâmetro Rest

O parâmetro rest (...) no JavaScript permite agrupar um número variável de argumentos em uma única variável. Ele é especialmente útil em funções que recebem um número indefinido de argumentos, tornando o código mais flexível e legível.

Confira os comentários no código e o Console através do Inspecionar (F12) para entender o conteúdo! ^-^
