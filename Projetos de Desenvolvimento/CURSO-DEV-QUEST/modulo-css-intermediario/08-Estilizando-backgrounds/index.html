Estilizando backgrounds

Background image

Aqui temos como exemplo o Background-image com largura e altura e o fundo verde, porém não irá aparecer pois o mesmo está coberto pela imagem.

Background repeat

Aqui temos como exemplo o Background-image com altura e com o Background-repeat: no-repeat, pois sem essa opçãop a imagem ia se repetir no eixo X e Y varias vezes.

Background size cover

Aqui temos como exemplo o Background-image com a opção de Background-size: cover, pois assim a imagem irá cobrir todo o conteudo do espaço da tela, lembrando que você deve dar uma altura para ela.

Background fixed

Aqui temos como exemplo o Background-image com a opção de Background-attachment: fixed, dessa forma irá criar o efeito parallax e a imagem ficará grudada na tela e o conteúdo ira rolar para baixo normalmente. Lembrando que você deve dar uma altura para ela.

Background position

Aqui temos como exemplo o Background-image com a opção de Background-position: bottom right com a opção de no-repeat também ativa. Lembrando que você deve dar uma altura para ela e colocar o no-repeat.

Background Shorthand

Aqui temos como exemplo o Background com a opção de SHORTHAND, ou seja a versão resumida de atributos e valores do seletor Background. Colocando tudo em apenas uma linha, respeitando a seguinte ordem: color, imagem url, repeat e position. Neste caso: background: black url('./images/cat.png') no-repeat right top;
