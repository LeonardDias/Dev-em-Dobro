GRID - AUTO FILL - AUTO FIT

Aqui iremos aprender sobre o AUTO FILL e o AUTO FIT, vamos lá, criaremos primeiro a base HTML e CSS inicial.
Com essas PROPRIEDADES não haverá a necessidade de CRIAR MEDIAS QUERIES para os GRIDS, ficará de forma mais simples.

Basicamente ambos irão se comportar da mesma forma no RESPONSIVO, alinhando-se conforme a tela fica menor. Porém quando a tela ficar maior notará a diferença.

AUTO-FILL

Aqui iremos ver como se comporta o AUTO-FILL. O comportameno de ambos são iguais em telas MENORES (RESPONSIVO), porém quando a tela ficar maior aqui no AUTO-FILL irá respeitar o tamanho máximo delimitado no MINMAX, que neste caso é 200px, ou seja ira gerar espaços livres no final do CONTAINER.
