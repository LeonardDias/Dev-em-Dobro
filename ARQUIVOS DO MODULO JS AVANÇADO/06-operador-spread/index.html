Operador Spread

O operador spread (...) no JavaScript permite expandir elementos de arrays, objetos ou strings em lugares onde múltiplos valores ou propriedades são esperados. Ele é útil para simplificar operações como copiar ou combinar estruturas de dados. Vamos ver como funciona em diferentes contextos!
