Flex Container - Align content

O align-content é uma propriedade que controla como as linhas de itens flexíveis são alinhadas dentro de um flex container quando há espaço extra no eixo vertical (altura) disponível. Isso é relevante quando você tem várias linhas de itens flexíveis dentro do seu contêiner.

Neste caso o ALIGN CONTENT ele irá alinhar os FLEX ITENS no espaço que sobra dentro do CONTAINER (EM LINHA) ficando completamente preenchido a linha do CONTAINER.

E é bastante parecido com JUSTIFY CONTENT, só que agora é no EIXO TRANSVERSAL (VERTICAL).

Este valor alinha as linhas de itens flexíveis no início do container.
