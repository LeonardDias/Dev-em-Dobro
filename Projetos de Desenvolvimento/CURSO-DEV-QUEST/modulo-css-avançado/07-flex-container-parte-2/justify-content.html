Flex Container - Justify content

A propriedade justify-content é usada para alinhar os itens (itens flexíveis) dentro de um contêiner flexível ao longo do eixo principal (main axis). O eixo principal pode ser horizontal (quando a flex-direction é row) ou vertical (quando a flex-direction é column). Existem várias opções de alinhamento disponíveis para a propriedade justify-content.

Flex-start

Alinha os itens no início do eixo principal.
