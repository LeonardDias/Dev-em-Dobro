Um Filme do Batman

Aqui iremos apender a introdução de RESPONSIVIDADE e MEDIA QUERY, iremos fazer esse site diminuir conforme as telas de celulares, para mod de didática criamos o site primeiro e depois o responsivo, mas lembre-se do MOBILE FIRST. Crie pensando primeiro no CELULAR e depois para PC.

Aqui também colocamos a imagem responsiva usando a TAG PICTURE e configurando com a TAG SOURCE (onde escolhemos o caminho e o tamanho da imagem menor) e depois com uma IMG para a imagem grande.

Temos a tabela do W3C com relação aos MEDIA QUERY, segue abaixo a mesma:

  1. @media (max-width: 375px) { }// dispositivos extra pequenos
  2. @media (max-width: 576px) { } // dispositivos pequenos (celulares em posição paisagem, menos de 768px)
  3. @media (max-width: 768px) { } // dispositivos médios (tablets, menos de 992px)
  4. @media (max-width: 992px) { } // dispositivos grandes (desktops, menos de 1200px)
  5. @media (max-width: 1200px) { } // dispositivos extra grandes (desktops grandes) // 1200px ou mais, não tem breakpoint

Neste caso apenas configuraremos a de 576PX, porém o ideal é você criar uma CSS de MEDIA QUERY por tela e linkalas separadas, pois assim fica muito mais fácil e de configurar.
